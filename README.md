# Expense Tracker (Prototype) 💰

⚠️ This project is currently a prototype and is still under active development.

A Spring Boot based Expense Tracker web application that allows users to manage their daily expenses securely.

## 🚀 Current Features

- User Registration & Login (Spring Security)
- Add, Update, Delete Expenses
- View all expenses
- Basic secure authentication system
- Simple UI using Thymeleaf

## 🛠 Tech Stack

- Java
- Spring Boot
- Spring Security
- Thymeleaf
- Maven
- HTML & CSS

## 📌 Project Status

This is an early-stage prototype.  
More features, improvements, and optimizations are being added.

## 🔮 Planned Improvements

- Expense categories
- Dashboard with charts & analytics
- Better UI design
- Export to CSV
- REST API support
- Database optimization
